## Feature Walkthrough

### Dashboard
After login you land on the **Dashboard**, which shows:
- **4 summary cards** — Total Minting count, Minting Volume, Total Redeem count, Redeem Volume
- **Minting by Status** — breakdown with progress bars (Pending, Under Review, Approved, Rejected, Processing, Completed, Failed)
- **Redeem by Status** — breakdown with progress bars (Pending, Processing, Completed, Failed)
- **Quick Actions** — one-click links to pending/under-review queues
- **System Status** — live indicator showing network and token info
- **Recent Activity** — last 10 transactions across minting and redeem with requester name, amount, and status

### Minting
Navigate to **Minting** via the sidebar. This page shows all USDX minting requests with:
- Full-text search by requester name or email
- Filter by status
- Date range filter
- Sortable columns
- **Approve / Reject / Start Review** actions on eligible requests
- CSV export of the current view
- Detail modal with full transaction info, proof of transfer link, and notes

### Redeem
Navigate to **Redeem** via the sidebar. This is a read-only view of redemption requests:
- Same search, filter, sort, and export capabilities as Minting
- Detail modal showing bank account, wallet address, and transaction hash

### Profile
Navigate to **Profile** via the sidebar or the top-right user dropdown. Shows:
- Account name, email, and role badge
- Member since and last login timestamps
- Security section (password change is disabled in the demo)

### Logout
Click your name in the top-right corner → **Logout**.

## What Is Mock vs. Production

| Feature | Current (Mock) | Production |
|---------|---------------|------------|
| Authentication | Single hardcoded user | Multi-user, JWT/OAuth |
| Data | MSW in-memory | Real API backend |
| Approve/Reject | Updates local state only | Triggers blockchain/bank workflows |
| CSV Export | Current page data only | Full dataset server-side export |
| Proof of Transfer | Fake Etherscan links | Real on-chain transaction links |
| Email notifications | None | Sent on status changes |
| RBAC | None | Role-based access per operator |
