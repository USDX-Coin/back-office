# Components

Shared, reusable components used across multiple features.

## Structure

```
components/
  ├── ui/            # shadcn/ui primitives — DO NOT EDIT MANUALLY
  ├── layout/        # App shell: Navbar, Sidebar, MainLayout, AuthGuard
  └── DataTable.tsx  # Generic data table (server-side pagination/sort/filter)
```

## Rules

- **`ui/`** — Generated by `npx shadcn@latest add <component>`. Do not modify these files directly. Customizations go in consuming components or via Tailwind theme tokens in `src/index.css`.
- **`layout/`** — App-wide layout components. `MainLayout` renders `Navbar` + `Sidebar` + `<Outlet />`. `AuthGuard` provides `ProtectedRoute` and `PublicRoute` wrappers.
- **`DataTable`** — Generic table accepting `ColumnDef<T>[]`, data, rowCount. Features: search, status filter, date range, sort, pagination, CSV export, loading skeletons, empty states. All filter state persisted in URL search params.

## DataTable Props

| Prop | Type | Description |
|------|------|-------------|
| `columns` | `ColumnDef<T>[]` | TanStack Table column definitions |
| `data` | `T[]` | Current page data |
| `rowCount` | `number` | Total row count (for pagination) |
| `isLoading` | `boolean` | Shows skeleton rows |
| `statusOptions` | `{value, label}[]` | Status filter dropdown options |
| `onExportCsv` | `() => void` | CSV export callback |
