# USDX Back-Office

Phase 1 frontend back-office for USDX stablecoin admin panel.

## Stack

Vite + React + TypeScript + TailwindCSS + shadcn/ui + TanStack (Router, Query, Table) + RHF + zod + Zustand + MSW.

## Dev

```bash
pnpm install
pnpm dev
```

App di `http://localhost:5173`.

Mock login: `admin@usdx.id` / `admin123`.
